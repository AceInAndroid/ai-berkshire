# A股去杠杆后的建仓与风险偏好监测

> **研究日期：** 2026年7月17日<br>
> **数据状态：** 盘中研究，主要指数采用约14:28延时行情<br>
> **适用情形：** 当前空仓，评估去杠杆大跌后的分批建仓时机<br>
> **核心观点：** 市场已经出现稳定资金承接，但融资资金、市场广度、成长风格和外盘科技尚未共同确认，现阶段更适合条件触发式分批建仓，而不是一次性重仓猜底。

---

| 指数 | 盘中涨跌 | 从盘中低点反弹 |
|---|---:|---:|

| 深证成指 | -3.75% | — |
| 创业板指 | -4.71% | +3.88% |

| 中证500 | -3.94% | — |
| 中证1000 | -4.40% | — |

市场情绪指标约为3，处于极弱水平。宽基ETF已有明显逆势净流入，但融资余额仍处于连续下降阶段。

因此，当前更接近：

> **有稳定资金承接，但风险偏好尚未真正回来。**

宽基ETF抄底资金与风险偏好资金不是同一种资金。前者可能来自长期配置、被动申购或稳定市场资金；后者通常表现为创业板、科创板、中证1000重新跑赢沪深300，市场上涨家数明显扩大。

## 第二步：三阶段建仓策略

### 第一阶段：观察仓，5%—15%

适用条件：

- 7月17日收盘没有再次接近盘中最低点；
- 下一个交易日不跌破7月17日低点；
- 跌停数量明显减少；
- 宽基ETF继续净流入；
- 外盘半导体不再出现连续暴跌。

这一阶段优先考虑：

1. 沪深300、中证A500等宽基指数；
2. 低负债、高现金流、高ROE公司；
3. 估值合理的红利和防御资产；
4. 基本面清晰、竞争优势稳定的行业龙头。

第一笔仓位暂时回避：

- 融资余额占流通市值比例特别高的股票；
- 连续上涨后刚开始补跌的AI、半导体小盘股；
- 高换手、纯题材、小市值股票；
- 跌停打开但没有基本面支撑的股票。

如果下一个交易日跌破7月17日低点，观察仓不继续加仓。

### 第二阶段：确认仓，增加到30%—50%

需要同时看到以下三个方面改善。

#### 1. 市场不再创新低

至少连续两个交易日：

- 上证、创业板、科创50不再跌破7月17日低点；
- 指数收盘位置处于当日振幅上半区；
- 回调缩量，反弹放量；
- 5日均线开始走平，不再陡峭向下。

#### 2. 市场广度明显修复

比较理想的确认标准：

- 上涨股票超过3000只；
- 上涨/下跌家数比超过1.5；
- 全市场涨跌幅中位数达到+1%以上；
- 跌幅超过5%的股票下降到300只以下；
- 跌停股票下降到50只以下；
- 创业板、科创板20%跌停股票下降到个位数；
- 涨停数量至少达到跌停数量的3倍。

#### 3. 高Beta资产重新跑赢

重点观察：

```text
创业板指 ÷ 沪深300
中证1000 ÷ 沪深300
```

如果两个相对强弱比值连续三个交易日上升，说明资金开始从大盘防御资产转向成长和小盘资产，风险偏好进入实质修复阶段。

### 第三阶段：趋势仓，增加到60%—80%

需要看到杠杆去化基本结束：

1. 融资余额连续2—3天不再明显下降；
2. 融资买入额占市场成交额的比例开始回升；
3. 高融资股票不再显著跑输；
4. 创业板、科创50重新站上5日或10日均线；
5. 外盘半导体和亚洲科技股完成止跌；
6. 市场出现新的基本面主线，而不只是超跌反弹。

在上述条件出现之前，不建议使用融资或其他杠杆。

## 第三步：证明“资金回来”的六个指标

### 指标一：融资余额下降速度

截至7月16日，融资余额单日减少约286.53亿元，并且已经连续11个交易日下降。

| 融资余额变化 | 市场含义 |
|---|---|
| 单日继续减少200亿—400亿元以上 | 去杠杆仍在进行 |
| 单日下降缩小到50亿元以内 | 抛售压力开始缓和 |
| 连续2—3天持平或小幅增加 | 杠杆资金开始稳定 |
| 融资余额温和增加且指数上涨 | 风险资金重新入场 |

最健康的变化不是融资余额突然大增，而是先停止快速下降，再出现温和回升。

还需要同时观察融资买入额和融资偿还额。如果融资余额稳定只是因为成交萎缩，而融资买入额仍处于低位，也不能认为风险偏好已经回来。

### 指标二：ETF流入后，指数是否停止创新低

近期股票ETF出现明显逆势净流入：

- 7月16日股票ETF资金净流入约403亿元；
- 过去一周股票ETF累计净流入约1340亿元；
- 资金主要流向沪深300、中证500、中证1000和科创50等宽基ETF。

有效承接的组合是：

```text
ETF持续净流入
+
指数不再创新低
+
跌停家数下降
```

无效承接的组合是：

```text
ETF持续净流入
+
指数继续创新低
+
融资余额继续快速下降
```

因此，ETF净流入金额必须与价格反应、融资余额和跌停结构结合判断。

### 指标三：成交额结构

成交额不是越大越好，需要区分放量下跌、缩量企稳和放量上涨。

#### 风险尚未释放完

- 指数继续下跌；
- 成交额持续维持高位；
- 跌停数量增加；
- 高融资股票继续领跌。

#### 初步企稳

- 指数不再创新低；
- 回调时成交额下降20%—30%；
- 跌停数量减少；
- 高融资股票不再领跌。

#### 资金真正回来

- 上涨日成交额高于前一交易日；
- 上涨股票数量明显增加；
- 半导体、券商、消费、成长等进攻板块放量上涨；
- 成交不再只集中于银行、电力、红利和其他防御资产。

### 指标四：市场广度

每天记录：

1. 上涨家数；
2. 下跌家数；
3. 涨停家数；
4. 跌停家数；
5. 跌幅超过5%的股票数量；
6. 全市场涨跌幅中位数。

风险偏好修复的最低标准是：

> **连续两天上涨股票超过下跌股票，并且全市场涨跌幅中位数转正。**

更强的确认信号是：

> **上涨股票超过3500只、市场中位数超过+1%、跌停少于50只。**

如果指数上涨，但仍有3000多只股票下跌，通常只是权重股支撑，不代表风险偏好全面回归。

### 指标五：成长和小盘的相对强弱

风险资金通常按照以下顺序回归：

```text
宽基大盘ETF
→
行业龙头
→
创业板、科创板
→
中证1000、小微盘
→
高换手题材股
```

| 相对表现 | 判断 |
|---|---|
| 沪深300强、创业板弱 | 仍处于防御和稳定阶段 |
| 创业板与沪深300同步 | 风险偏好开始修复 |
| 创业板、中证1000连续跑赢沪深300 | 风险偏好明显回来 |
| 小微盘上涨但大盘和成交恶化 | 可能只是短线题材炒作 |

当前创业板、科创50和中证1000仍显著跑输沪深300，暂时不能确认风险偏好已经恢复。

### 指标六：外盘半导体能否止跌

这次下跌存在明显的全球科技链冲击，需要观察：

- 费城半导体指数；
- 纳斯达克指数；
- 日经225；
- 台湾加权指数；
- 韩国KOSPI及主要半导体公司；
- 恒生科技指数。

比较可信的外盘止跌信号：

1. 费城半导体指数不再创新低；
2. 出现一次约3%以上的有效反弹；
3. 第二个交易日没有完全回吐；
4. 日本、台湾、韩国科技股同步企稳；
5. A股半导体不再跟随外盘创新低。

如果外盘半导体继续每日大幅下跌，A股科技股即使盘中反弹，也很难立即形成稳定趋势。

## 第四步：区分护盘资金和风险偏好资金

### 护盘或稳定资金回来

常见表现：

- 沪深300ETF、中证A500ETF大额净流入；
- 银行、电力、运营商、石油上涨；
- 上证指数跌幅相对较小；
- 创业板、科创板继续明显下跌；
- 上涨家数明显少于下跌家数。

护盘资金可以降低系统性风险，但不意味着高Beta股票马上上涨。

### 风险偏好资金回来

常见表现：

- 创业板、科创50、中证1000跑赢沪深300；
- 券商、半导体、软件、消费电子等进攻板块上涨；
- 上涨家数超过3000只；
- 跌停数量快速下降；
- 高融资股票不再跑输；
- 融资买入额恢复；
- 上涨日成交额放大。

目前市场更接近第一种：有资金稳定指数，但风险偏好仍然很低。

## 第五步：七项确认表

每天收盘后打分，每满足一项得1分。

| 指标 | 确认条件 |
|---|---|
| ① 指数结构 | 连续两天不跌破7月17日低点 |
| ② 市场广度 | 上涨家数连续两天超过下跌家数 |
| ③ 跌停结构 | 跌停少于50只、20%跌停少于10只 |
| ④ 融资数据 | 单日融资余额降幅缩小到50亿元以内 |
| ⑤ 相对强弱 | 创业板/沪深300连续三天上升 |
| ⑥ 外盘环境 | 费城半导体不再创新低并出现有效反弹 |
| ⑦ 资金承接 | 宽基ETF持续流入且指数不再创新低 |

| 得分 | 市场状态 | 空仓者策略 |
|---:|---|---|
| 0—2分 | 风险释放中 | 保持现金，或不超过10%观察仓 |
| 3—4分 | 初步企稳 | 分批增加到20%—40% |
| 5—6分 | 风险偏好修复 | 增加到50%—70% |
| 7分 | 趋势确认 | 根据估值与公司质量继续配置 |

按照7月17日盘中信息，市场约为2分：

- 宽基ETF存在资金净流入；
- 指数从盘中最低点明显反弹；

但同时：

- 市场广度仍然极弱；
- 融资余额还在快速下降；
- 成长和小盘继续跑输；
- 外盘科技尚未企稳；
- 市场情绪指标仍处于极低水平。

因此，当前仍属于恐慌后的承接阶段，而不是趋势已经确认。

## 第六步：配置顺序

### 第一批：低风险核心仓

- 沪深300、中证A500等宽基指数；
- 低负债、高现金流、高ROE龙头；
- 估值没有明显泡沫的红利资产。

### 第二批：基本面成长仓

- 经过估值消化的科技龙头；
- 有真实利润、订单和现金流的半导体、AI基础设施公司；
- 产业趋势未被破坏，但股价因流动性冲击下跌的公司。

### 第三批：高弹性仓位

- 中证1000、小盘成长；
- 高弹性科技；
- 高融资、高换手股票。

去杠杆阶段不宜将配置顺序反过来。第一笔仓位直接买入高融资、高波动的小盘科技，会同时承担行业、估值、流动性和杠杆四重风险。

## 第七步：最终行动清单

### 当前空仓投资者

1. 不在暴跌当天一次性重仓；
2. 如需参与，观察仓控制在5%—10%；
3. 等待收盘和下一个交易日确认7月17日低点是否有效；
4. 等待7月17日两融结算数据，观察融资余额是否异常大降；
5. 七项确认表达到3—4分后再明显增加仓位；
6. 七项确认表达到5分以上，才考虑明显提高成长和小盘配置；
7. 在去杠杆确认结束前不使用融资。

### 暂停加仓或降低风险的条件

- 指数跌破7月17日盘中低点；
- 融资余额继续每日下降200亿—400亿元以上；
- 跌停数量重新增加；
- 高融资股票继续显著跑输；
- 费城半导体及亚洲科技市场继续创新低；
- ETF大量流入但指数仍无法企稳。

### 可以提高仓位的条件

- 指数连续两个交易日不创新低；
- 上涨家数连续两天超过下跌家数；
- 融资余额下降速度显著放缓；
- 创业板、中证1000相对沪深300转强；
- 外盘半导体完成有效止跌；
- 上涨日放量、回调日缩量。

---

## 最终判断

> **真正值得买入的信号，不是“已经跌了很多”，而是抛盘开始减少、融资余额停止快速下降、市场广度转正、成长重新跑赢，而且外盘不再持续制造新风险。**

当前空仓的最大优势是拥有选择权。在资金承接已经出现、但风险偏好尚未确认的阶段，采用条件触发式分批建仓，比一次性押注最低点更有利于控制回撤和判断错误成本。
